# Automated Kubernetes Deployment using Jenkins

This repository demonstrates an end-to-end Jenkins CI/CD pipeline that builds,
containerizes, and deploys a Hello World application to Kubernetes using Terraform.

## Structure
- app/: Application source
- infra/: Terraform IaC
- k8s/: Kubernetes manifests
- Jenkinsfile: CI/CD pipeline
